# Calcium_likelihoods

Package dependencies may cause issues so use the environment file environment.yml. Code to import environment variable in conda is

```
conda env create -f environment.yml
```

## Examples

We include example scripts and interactive Colab examples. 
- **Calcium HMM Simulation** - A Colab running the Calcium HMM Simulation in Figure 2 can be found [here](https://colab.research.google.com/drive/1AKFzuVUec5zbFK2zcH2Ztjv4d3AfNQRJ?usp=sharing) 
- **Calcium LFADS Simulation** - A Colab running the Calcium LFADS Simulation can be found[here](https://colab.research.google.com/drive/1hRFmKf4t5xQbAvxPcGkS_AsfxTJqdbev#scrollTo=cJ9Av2ad73g5). This Calcium LFADS implementation is in https://github.com/davidzoltowski/computation-thru-dynamics.

Environment name is "ca_like". The jupyter notebook example files will show the calcium likelihood in use for latent dynamical models.
